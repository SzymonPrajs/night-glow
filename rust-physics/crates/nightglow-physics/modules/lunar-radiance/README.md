# Lunar radiance

Owns reflected solar and earthshine-driven spectral radiance across the apparent lunar disk.

Inputs: Sun–Moon–observer geometry, lunar orientation/distance, incident solar spectrum, earthshine irradiance, photometric/topographic model, wavelength basis. Outputs: disk-resolved or integrated lunar radiance and phase/quality diagnostics.

Research must cover lunar BRDF/photometric functions, opposition surge, terrain/albedo variation, libration, finite solar disk, phase reddening, eclipses, and validation against calibrated lunar irradiance. A scalar phase curve is only a fallback. Planet light is normally negligible for lunar illumination and should be included only if an error budget demonstrates relevance.
