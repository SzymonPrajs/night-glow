# Molecular atmosphere

Owns vertical thermodynamic/composition state and wavelength-dependent molecular extinction, scattering, and absorption.

Inputs: altitude/geopotential profile, pressure, temperature, humidity, gas composition, wavelength basis, gravity/Earth geometry. Outputs: number densities, refractive index support, Rayleigh coefficients/phase function, absorption coefficients, and optical-depth diagnostics.

Research must cover dry/moist refractivity, depolarization, ozone and relevant molecular absorption, standard-atmosphere fallbacks, weather-profile interpolation, spherical stratification, and uncertainty. Validation includes hydrostatic consistency, Beer–Lambert limits, wavelength scaling where applicable, and comparison with a line-by-line/band reference.
