# Earth orientation

Owns precession-nutation, Earth rotation angle/sidereal quantities, polar motion, and transformations driven by Earth-orientation parameters.

Research must choose IAU model versions, EOP ingestion/interpolation, prediction/out-of-range policy, and update cadence. Reference and interactive modes may cache matrices over bounded intervals only after angular error is measured. Validation spans long date ranges, leap boundaries, polar motion, and comparison with SOFA vectors.
