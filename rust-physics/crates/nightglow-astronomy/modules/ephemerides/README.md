# Ephemerides

Owns loading/interpolating a selected planetary/lunar ephemeris and returning barycentric/geocentric states with declared time/frame semantics.

Research must compare direct support for JPL DE440/DE441 or an independently validated compact representation; data size and browser tiling; interpolation order/error; light-time iteration; relativistic corrections; and supported date range. Native precompute may create compact interval coefficients, but must retain the upstream revision and measured positional error.
