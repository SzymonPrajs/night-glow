# Solar-system apparent geometry

Owns apparent observer-relative Sun, Moon, and planet directions, distances, angular radii, illuminated fractions, phase angles, orientation/libration, rings, occultations, eclipses, and terrain/horizon query inputs.

It consumes ephemeris and frame states and produces geometry only. Body radiance belongs to the corresponding physics source modules. Research must include finite light time, aberration/deflection policy, topocentric lunar parallax, apparent disk overlap, and cadence/error bounds for cached interactive updates.
