# Solar radiance

Owns top-of-atmosphere solar spectral radiance/irradiance and resolved-disk effects.

Inputs: Sun–observer distance and apparent disk geometry from astronomy, solar spectrum revision, wavelength basis, optional activity/eclipsing state. Outputs: spectral disk radiance or directional irradiance, limb-darkening representation, and uncertainty.

Research must select an absolute reference spectrum, distance scaling, limb darkening, finite disk integration near horizon, eclipse/occultation, and spectral sampling needed for twilight absorption. Display bloom is explicitly not solar physics.
