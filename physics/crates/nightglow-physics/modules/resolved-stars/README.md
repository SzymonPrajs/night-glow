# Resolved stars

Owns intrinsic/apparent stellar spectral flux, magnitude conversion, variability, and representation as point or finite-angular sources. Astronomy owns propagated positions/distances.

Inputs: calibrated catalogue photometry/astrophysical parameters, propagated distance/state, extinction policy, variability state, wavelength basis. Outputs: top-of-atmosphere spectral flux/radiance source records and quality flags.

Research must cover Gaia passbands and zero points, missing/poor BP/RP data, bright-star saturation/completeness, binary/variable sources, spectral-library fitting, interstellar extinction, and flux-conserving LOD. Catalogue stars removed into the resolved tier must be removed from diffuse integrated starlight to prevent double counting.
