# Planetary radiance

Owns reflected/emitted spectral radiance of solar-system planets after astronomy supplies disk, phase, distance, and orientation.

Inputs: body, phase geometry, angular size, reference spectra/albedo/phase law, wavelength basis. Outputs: disk-integrated or resolved spectral radiance and uncertainty.

Research includes body-specific phase functions, rings, thermal contribution where the observation basis requires it, variability, and bright-planet PSF behavior. Planetary illumination of Earth/Moon is expected to be negligible for normal night-sky radiance but should be quantified rather than assumed into a major feedback loop.
